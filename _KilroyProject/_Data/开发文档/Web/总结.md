# 总结

```
Web商用免费字体
* serif, sans-serif, monospace, fantasy, cursive
* -apple-system-font
* '黑体', '宋体', '仿宋', '楷体'
* '思源黑体'
```

```
开启谷歌浏览器跨域模式
open -n /Applications/Google\ Chrome.app / --args --disable-web-security  --user-data-dir=/Users/kilroy_lyk/Applications/Chrome\ Apps.localized
```

```
document.designMode = 'on'; 可以随意编辑网站
```

```
Less 如果不需要转义的字符串，要写成~'X'，如果为空则要写成，例如cale(1/1)
```

```
YUI Compressor JS 压缩的数组不能有特殊字符串的key
```

```
YUI Compressor JS 压缩的方法不能在()内设置默认值，否则压缩失败
```

```
YUI Compressor CSS 不能出现0s、0%等，会被压缩为0,需要使用from或cale(0s)替换
```

```
form onclick="return false;" 会导致内部所有的默认事件失效
```

```
<a>的javascript:void(0);会导致默认点击事件失效，比如radio
```

```
移动端img标签长按无反应添加-webkit-touch-callout: default;
```

```
上传内容过大时。需要考虑丢包导致的解密失败，需要将数据包在FormData里
```

```
html2canvas的问题
scale与设备不匹配可能导致生成空白
提高清晰度需要增加dom节点的比例
外部图片需要允许跨域或获取Base64
不支持transform
```

```
微信调起App，按钮隐藏再显示后，节点会失效，需要保持按钮一直在页面内显示
```

```
Less循环
.loop_li(@i) when (@i<=11) {
  &:nth-child(@{i}) i {
    .bg('profession_@{i}.png', 40*@px, 40*@px, true);
  }
  .loop_li(@i+1);
}
.loop_li(1);
```
